# Stackovervibe — Контекст проекта

## Название
Stackovervibe (игра слов: Stack Overflow + Vibe)

## Домен
stackovervibe.ru (куплен)

## VPS
Текущий сервер (где ведётся разработка)

## Кто заказчик
Иван Марцинкевич (@CheatB) — новоиспечённый вайбкодер, создаёт приложения с помощью нейросетей.

## Зачем
- Прошёл через боль настройки окружения для вайбкодинга (метод проб и ошибок)
- Хочет сэкономить эту боль другим
- Собрал фреймворк работы и кучу примочек, но контента по нему пока нет
- Цели: SEO → трафик → монетизация → вирусность

## Что за продукт
Сайт-база знаний по вайбкодингу:
- Наработки, инструменты, конфиги, опыт
- Место где есть ответы на все вопросы без лишнего шума от ИИ
- В перспективе: UGC — другие пользователи выкладывают свои наработки (с модерацией на вредоносность)

## Для кого
Ребята, залетающие в вайбкодинг:
- Хотят настроить среду без боли
- Хотят понять что делать и как
- Нужны конкретные ответы, а не вода

## Стек
На выбор архитектора, подберём под задачу после brainstorming.
