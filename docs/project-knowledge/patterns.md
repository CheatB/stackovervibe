# Stackovervibe — Паттерны проекта

## Дизайн-система
- **Тёмная тема** — основа, светлой темы нет
- **Ретро-хакерский стиль 80-х** — Hackerman из Kung Fury
- Визуальные элементы: зелёные терминалы, глитч-эффекты, ASCII-арт, неон, CRT-свечение
- Клише из фильмов про хакеров — "на приколе" с визуальными отсылками
- Красота и стильность — приоритет наравне с функциональностью
- Дизайн будет проработан через frontend-design скилл отдельной сессией

## Контент-паттерны
- Язык: русский, без канцелярита, живо
- Каждый инструмент — отдельная страница (SEO: одна страница = один запрос)
- Каждая страница инструмента содержит: описание, зачем нужен, как настроить, скачать конфиг
- Гайды в "пути новичка" идут в строгом порядке, со ссылками на предыдущий/следующий

## SEO-паттерны
- SEO — главная стратегия привлечения трафика
- Каждая страница оптимизирована под конкретные поисковые запросы
- Meta теги, Open Graph, structured data, sitemap
- Семантическая вёрстка, быстрая загрузка

## Git workflow
- Conventional commits: feat/fix/refactor/docs/test/chore
- Коммиты только по команде владельца
- Дефолтная ветка: main

## Контент-процесс
- Автор: Иван + ИИ (суть → оформление → доводка ToV)
- Публикация через CMS-панель
- Фреймворк переписывается — контент обновляется по мере готовности
