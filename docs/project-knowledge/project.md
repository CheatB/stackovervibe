# Stackovervibe — Обзор проекта

## Что это
Структурированная база знаний по вайбкодингу. Название — игра слов: Stack Overflow + Vibe.

## Проблема
Информация о вайбкодинге разбросана по YouTube, Telegram, Reddit, ChatGPT — нигде нет системы. Люди тратят кучу токенов и времени, а результат не оправдывает ожиданий. Настроить рабочее окружение — боль и метод проб/ошибок.

## Аудитория
1. **Основная:** вайбкодеры, которые уже попробовали и обожглись (токены → мусор, время → ненастроенный процесс)
2. **Вторичная:** новички, только входящие в вайбкодинг

## Ключевые фичи MVP
- **Лендинг** — ретро-хакерский стиль 80-х, обзор фреймворка
- **Путь новичка** — линейный маршрут от нуля до рабочей среды
- **Каталог инструментов** — отдельная страница на каждый инструмент (скиллы, хуки, команды, правила) с возможностью скачать конфиги
- **CMS-панель** — публикация и редактирование контента через интерфейс
- **SEO-оптимизация** — каждая страница заточена под поисковую выдачу

## Scope — что НЕ входит в MVP
- Английская версия
- UGC (контент от других пользователей)
- Q&A формат (как Stack Overflow)
- Видео и интерактивные примеры кода
- Монетизация и авторизация пользователей
